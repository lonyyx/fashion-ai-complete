# FashionAI - Умный поиск одежды

Fullstack веб-приложение для интеллектуального поиска одежды с использованием AI.

## Технологии

- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js, Vercel Functions
- AI: Hugging Face Inference API
- Хостинг: Vercel

## Функциональность

- Умный анализ запросов на естественном языке
- Подбор товаров из популярных магазинов
- AI-помощник с рекомендациями
- Адаптивный дизайн

## Локальная разработка

```bash
npm install
vercel dev
